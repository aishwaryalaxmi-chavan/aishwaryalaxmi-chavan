## 🚀 Featured Projects

### 1. LLM-Assisted OFAC Sanctions Screening (SentinelRAG)
Built an OFAC SDN sanctions screening pipeline using **BM25 + embeddings + cross-encoder** retrieval. Designed a deterministic decision layer for auditable **MATCH / NO MATCH** outcomes and improved system efficiency with **56% latency reduction** while maintaining **99% recall**.

**Tech:** Python, Sentence-Transformers, ChromaDB, Cross-Encoder

### 2. AI Semantic Book Recommendation System
Built a semantic recommendation system using **transformer embeddings**, **vector similarity search**, and a **retrieval + ranking pipeline**. Exposed the system through a **Gradio dashboard** and benchmarked it against keyword-based search on **7k+ book descriptions**.

**Tech:** Python, Hugging Face Transformers, LangChain, ChromaDB, Gradio, Pandas

### 3. Real-Time Graph Data Processing Pipeline
Built a near real-time ingestion pipeline using **Kafka**, **Neo4j**, **Docker**, and **Kubernetes**. Implemented graph analytics such as **PageRank** and **BFS** and deployed containerized services for scalable, low-latency processing.

**Tech:** Python, Kafka, Neo4j (GDS), Kubernetes, Docker, Helm
